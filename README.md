# Page

A minimalist, monochrome personal profile page with an immersive dark aesthetic.

## Features

- Video background with grayscale filter
- Animated star field
- Interactive music player with visualizer
- 8-shade grayscale theme picker
- Timezone display with visitor comparison
- Smooth animations and hover effects
- Responsive design
